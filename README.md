CLI Tool for streaming YT audio
===============================

Dependencies
------------

[ffmpeg](https://ffmpeg.zeranoe.com/builds/)  
[Node Package Manager](https://www.npmjs.com/get-npm)  

### Test Dependencies

```powershell
ffmpeg -version
node -v
npm -v
```

Install
-------

Clone / Download repository

```powershell
cd <REPOSITORY>
npm install
```

Sample Run
----------

```powershell
cd <REPOSITORY>
node main.js https://www.youtube.com/watch?v=nTjc1sPktlY
```
